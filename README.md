# Food Delivery Data Analysis 

This repository contains an end-to-end data analysis project on a food delivery dataset using **Python, Pandas, SQL, and Jupyter Notebook**.

The project focuses on extracting business insights such as revenue trends, customer behavior, membership analysis, and restaurant performance.

---

##  Project Files

- **orders.csv**  
  Contains order-level details such as order ID, user ID, restaurant ID, order date, and total amount.

- **users.json**  
  Contains user-related information including membership type.

- **final_food_delivery_dataset.csv**  
  Final merged dataset created by joining orders, users, and restaurant data.

- **restaurants.sql**  
  SQL file containing restaurant-related data such as cuisine and ratings.

- **test.ipynb**  
  Jupyter Notebook with all analysis, SQL queries, and verification steps.

---

##  Tools & Technologies Used

##  Key Analysis Performed

- Total orders and revenue analysis
- Gold membership-based insights
- City-wise and quarter-wise revenue trends
- Rating-based restaurant performance
- Data validation and verification using Pandas & SQL
